# 📊 Excel Sales Dashboard

## 1. 🎯 Objective
Visualize and analyze sales data to monitor product performance, track regional sales, and highlight trends for business improvement.

## 2. 🛠 Tools & Technologies
- Excel (Pivot Tables, Charts, Formulas, VLOOKUP/XLOOKUP)  

## 3. 📊 Key Insights
- Top-selling products contribute majority of revenue.  
- Regional performance highlights areas for growth.  
- Sales trends show peak and low periods for planning.  

## 4. 🧠 Recommendations
- Target high-performing regions for promotions.  
- Focus on popular products while reviewing underperforming ones.  
- Adjust inventory and marketing according to sales trends.
