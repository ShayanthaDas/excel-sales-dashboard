# Excel Sales Dashboard

## Overview

This project is a basic sales dashboard built in Excel to understand overall sales performance and business trends. The main focus was to turn raw sales data into something visual and easy to understand.

I kept the analysis simple and focused on practical business insights.

## What this project is about

* Tracking overall sales performance
* Understanding profit and revenue trends
* Identifying top products and categories
* Creating a simple interactive dashboard in Excel

## Tools used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Basic data cleaning

## Dashboard features

* Total sales and profit summary
* Category-wise breakdown
* Product-level performance
* Time-based sales trends
* Filter options using slicers

## Key insights

* A few products generate most of the sales
* Profit is not evenly distributed across all categories
* Some products perform well in sales but not in profit
* Sales trends change over time based on category demand

This project helped me understand how Excel can be used as a simple but powerful tool for business analysis. I also learned how to present data in a clean and structured way.
