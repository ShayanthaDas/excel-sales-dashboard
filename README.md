# Sales Data Analysis Dashboard (Excel)

## Project Overview
Built a professional sales dashboard using Microsoft Excel to analyze sales performance.

## Tools Used
- Excel
- Pivot Tables
- Charts
- Formulas
- KPI Metrics

## Features
✔ Total Sales KPI
✔ Total Quantity KPI
✔ Average Sales
✔ Category-wise sales analysis
✔ Region-wise contribution (Pie chart)
✔ Sales trend over time (Line chart)
✔ Dashboard design

## Skills Demonstrated
Data Cleaning | Data Analysis | Dashboard Design | Reporting
